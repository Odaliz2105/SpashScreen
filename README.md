# Photo Gallery App

Aplicación móvil desarrollada con Ionic + Angular + Capacitor.

## Funcionalidades

- Captura de fotografías usando la cámara del dispositivo.
- Visualización de imágenes en la galería.
- Guardado de imágenes en almacenamiento local.
- Guardado automático en la galería del teléfono.
- Nombre personalizado para las fotografías usando apellido.
- Alerta personalizada en Tab1.
- Splash Screen personalizado.
- Ícono personalizado de la aplicación.

---

## Tecnologías utilizadas

- Ionic Framework
- Angular
- Capacitor
- TypeScript
- Android Studio

---

## Plugins utilizados

- @capacitor/camera
- @capacitor/filesystem
- @capacitor/preferences
- @capacitor/status-bar
- @capacitor/app
